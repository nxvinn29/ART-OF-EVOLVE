# Features

This directory contains the feature-based modules of the application.

## Structure

Each feature is organized as follows:
- `data`: Data layer (Models, Repositories, Data Sources)
- `domain`: Domain layer (Entities, Usecases)
- `presentation`: Presentation layer (Widgets, Providers, Controllers)

## List of Features

- **auth**: Authentication (Login, Sign Up)
- **home**: Dashboard and Home enhancements
- **self_care**: Journaling, Meditation, etc.
- **habits**: Habit tracking logic
- **gamification**: XP, Levels, and Badges
